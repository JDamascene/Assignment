The grid Info

Mr.Ephraim tought that in bootstrap, the grid system allocates twelve (12) collumns in each row which is specified by a row class name. It neatly arranges all items in a tabble format despite there being no border divisions just to make the website neat.

Important info

For the items of any website and or web page to be in a grid format, the web developer oughts to begin with a div of class row, s/he then needs to put another div of class col iside the row div. This should be accompanied by the desired device screen size, wheather medium, small or large, then the collumn span of which is a digit and all items are separated by ahyphen (-).
